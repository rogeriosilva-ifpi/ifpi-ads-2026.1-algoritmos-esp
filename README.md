# Algoritmos ESP - IFPI ADS 2026.1

Repositório de exercícios e atividades práticas da disciplina de Algoritmos ESP (Estruturação e Solução de Problemas) do curso de Análise e Desenvolvimento de Sistemas (ADS) do Instituto Federal do Piauí (IFPI), período 2026.1.

## Sobre o Projeto

Este repositório contém implementações em Python de exercícios básicos de programação, focando em:
- Entrada e saída de dados
- Operadores aritméticos
- Manipulação de variáveis
- Lógica de programação
- Estruturas de dados básicas

## Estrutura do Projeto

### Exercícios Introdutórios
- **hello.py** - Programa básico de saudação
- **saudacao.py** - Cálculo de idade com base no ano de nascimento
- **operadores_arimeticos.py** - Demonstração de operadores aritméticos em Python

### Exercícios de Lógica
- **sorteador.py** - Sorteio aleatório de números usando embaralhamento de lista

### Questões/Desafios
- **q3_min_em_h_min.py** - Conversão de minutos totais em horas e minutos
- **q5_cdu.py** - Decomposição de número em centenas, dezenas e unidades (CDU) e cálculo do somatório dos dígitos
- **q11_reverso.py** - Inversão de número de 3 dígitos usando decomposição CDU

## Como Executar

Certifique-se de ter o Python 3 instalado em seu sistema.

Para executar qualquer exercício:

```bash
python nome_do_arquivo.py
```

Exemplo:
```bash
python q3_min_em_h_min.py
```

## Tecnologias Utilizadas

- Python 3
- Biblioteca padrão: `random`, `math`

## Conceitos Abordados

- **Entrada de dados**: `input()`
- **Saída de dados**: `print()`, f-strings
- **Operadores aritméticos**: `+`, `-`, `*`, `/`, `//`, `%`
- **Conversão de tipos**: `int()`
- **Divisão inteira e módulo**: Decomposição de números
- **Manipulação de listas**: Embaralhamento com `random.shuffle()`
